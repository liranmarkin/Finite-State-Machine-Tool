Finite State Machine Tool


code instructions:
"[state name]:" -declares new state with name [state name] (don't forget nekudotayim at the end of the line)
"print [text]" -prints [text] on that state
"goto [state name]" -jumps to [state name]
"on [character] [state name]" -jumps to [state name] if the next character on input is [character]
