The big white box to the left is the FSM designer.  Here's how to use it:

  • Add a state: double-click on the canvas
  • Assign state name: click on a state and enter name
  • Add an arrow: shift-drag on the canvas
  • Assign letter to an arrow: click on an arrow and enter a letter (for state transition based on input)
  • Move something: drag it around
  • Delete something: click it and press the delete key (not the backspace key)
  • Make accept state: double-click on an existing state

The FSM compiles into code above.
